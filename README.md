# PORTAL_DE_TRAMITES_DEL_ESTADO

## Instalación del proyecto
```sh
npm install
```
### Compilación y Auto-Recarga para Desarrollo
```sh
npm run client
```
### Revisión de Tipados, Compilación y Minificación para Producción
```sh
npm run build
```
### Revisión con [ESLint](https://eslint.org/)
```sh
npm run lint
```
